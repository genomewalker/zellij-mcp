Get Started

Clone, install, restart.

1

Clone the repository

git clone https://github.com/genomewalker/zellij-mcp
2

Run the installer

cd zellij-mcp && ./scripts/install.sh

Installs Python dependencies and registers the MCP server.

3

Restart Claude Code

Then start talking: "open a floating pane", "create a new tab"

toggle_floatingToggle floating panes
new_tabCreate a new tab
focus_tabSwitch to tab by name/index
write_charsSend text to pane

session_infoGet current session details
